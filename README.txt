Инструкция:

1. Установите Python версии 3 и выше.
2. Установите Библиотеку keyboard (Откройте Командную строку и впишите "pip install keyboard" без скобок).
3. Установите Библиотеку tkinter (Откройте Командную строку и впишите "pip install tk" без скобок).
4. Запустите файл "Auto Key Presser.py".
5. Пользуйтесь.
